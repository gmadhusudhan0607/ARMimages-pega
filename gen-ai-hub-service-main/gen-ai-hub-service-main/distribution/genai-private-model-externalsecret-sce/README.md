# SCE Generation project

Produce an SCE asset that can be used by Cloud 2.x

This project uses the [com.pega.sce.publishing](https://git.pega.io/projects/PP/repos/gradle-prpc-platform-plugins/browse/projects/sce-plugin) gradle plugin.

To build locally
```bash
./gradlew publishToMavenLocal
```

See [com.pega.sce.publishing](https://git.pega.io/projects/PP/repos/gradle-prpc-platform-plugins/browse/projects/sce-plugin) for more tasks & configuration options.
